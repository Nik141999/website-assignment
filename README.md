# ELO - Frontend

## Requirements

- **React.js**: v18
- **Node.js**: v18
- **NPM** or **Yarn**

## Installation

### 1. Clone the Repository
```sh
git clone <repository-url>
cd <project-directory>
```

### 2. Install Dependencies
```sh
npm install
```
### 3. Run the Application

#### Start the Development Server
```sh
npm run dev
```
